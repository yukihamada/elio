=== App Review Information ===

【Notes for Review】

This is an AI chat application with multiple inference modes:

1. LOCAL MODE (Free)
- On-device AI inference using CoreML
- No internet required
- Works without any configuration

2. PRIVATE MODE (Free)
- Connects to trusted devices via Bonjour
- Requires local network permission
- User must manually trust devices

3. FAST MODE (1 token/message)
- Uses Groq API
- Requires user's own API key from groq.com

4. GENIUS MODE (5 tokens/message)
- Uses OpenAI/Anthropic/Google APIs
- Requires user's own API key

5. PUBLIC MODE (2 tokens/message)
- P2P connection to community servers

【How to Test】

For LOCAL MODE:
- Just open the app and start chatting
- No API key or account needed

For CLOUD MODES (Fast/Genius):
- Go to Settings > API Keys
- Enter your own API key from the provider
- Get free API keys from:
  - Groq: https://console.groq.com
  - OpenAI: https://platform.openai.com
  - Anthropic: https://console.anthropic.com
  - Google: https://makersuite.google.com

For P2P MODES:
- Requires two devices on the same network
- One device runs as server, other connects

【Demo Account】
Not required - Local mode works without any account.
